# **HTTP MESSAGES**

## **1. 报文的组成部分**
报文由3个部分组成:

1. 对报文进行描述的**起始行(start line)**;
2. 包含属性的**首部(header)**;
3. 可选的包含数据的**主体(body)**;

报文分成两类: 请求报文(request message) 和 响应报文(response message)
请求报文向web服务器请求一个动作，响应报文将结果返回给客户端;
请求报文和响应报文的基本结构相同

http报文各组成部分:
1. method 方法
客户端希望服务器对资源进行的操作
2. 请求url
命名了所请求的资源，或者url路径组件的完整url
3. 版本 versioin
报文使用的htpp的版本
4. 状态码
三维数字，描述了请求过程中所发生的情况 
5. 原因短语
数字状态吗的可读版本
6. 首部(header)
可以有0个或多个首部, 
7. 实体的主体部分(entity-body)
实体的主体部分包括一个有任意数据组成的数据块
